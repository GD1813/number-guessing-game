# Number Guessing Game (Python)

A simple console-based number guessing game built using Python.  
The player selects a difficulty level and tries to guess a randomly generated number within a limited number of attempts.

## Features
- Three difficulty levels: Easy, Medium, Hard
- Random number generation
- Limited attempts based on difficulty
- User-friendly hints (Too High / Too Low)
- Input validation

## Technologies Used
- Python
- Loops and Conditional Statements
- Functions
- Random module
- Exception Handling

## How to Run
1. Clone the repository or download the file
2. Run the program using: python main.py
